# Linux Commands

## Navigation
```bash
pwd        # print working dir
ls -la     # list with details
cd /path   # change dir
```

## File Operations
``` 
cp file1 file2    # copy
mv file1 file2    # move/rename
rm -rf dir/       # remove dir
```

## Sudo - set user
```bash
sudu -i           # login as root user
exit              # logout from the logged in user 
```
